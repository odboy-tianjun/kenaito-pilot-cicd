# Kubernetes 的包管理器

> 官方文档 https://helm.sh/zh/docs

# 开源地址

https://github.com/helm/helm

# 安装脚本（v3版本）

> https://helm.sh/zh/docs/v3/

```shell
curl -O https://get.helm.sh/helm-v3.20.2-linux-amd64.tar.gz
tar -xvf helm-v3.20.2-linux-amd64.tar.gz
mv linux-amd64/helm /usr/local/bin/helm
chmod +x /usr/local/bin/helm
rm -rf linux-amd64
```
